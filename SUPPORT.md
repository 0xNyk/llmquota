# Support

Use GitHub issues for reproducible bugs and scoped feature requests. Include the llmquota version,
operating system, Node version, affected provider, and sanitized output where useful.

Do not post tokens, credential files, account identifiers, private paths, or raw bus contents.
Security reports belong in a private GitHub security advisory, as described in
[SECURITY.md](SECURITY.md).

This is a maintainer-led project without a response-time guarantee. Provider collectors depend on
interfaces that may change without notice. A broken undocumented endpoint is handled on a
best-effort basis.
